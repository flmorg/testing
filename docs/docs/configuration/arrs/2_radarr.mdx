# Radarr Settings

<Note>
  Multiple instances can be specified for each *arr using this format, where `<NUMBER>` starts from 0:
```yaml
<ARR>__INSTANCES__<NUMBER>__URL
<ARR>__INSTANCES__<NUMBER>__APIKEY
```
</Note>
